# دليل رحلة إلى العلا
هذا المشروع يقدم خطة مقترحة لرحلة سياحية إلى العلا.
## معلومات الرحلة
- مدة الرحلة: 3 أيام
- الميزانية المتوقعة: 2000 ريال
- عدد المسافرين: 5 أشخاص
- وسيلة التنقل الرئيسية: سيارة
## أعضاء الفريق
- اسم القائد/ عبدالله طارق الغامدي
- اسم الطالب الثاني/محمد القحطاني 
- اسم الطالب الثالث /محمد العجمي 
-  اسم الطالب الرابع/عبدالرحمن العتيبي
